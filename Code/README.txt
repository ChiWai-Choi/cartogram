=============HOW TO RUN================
In a python interpreter, run the 
following file:

- DataDriver.py

You will be asked to decide which data 
set to run:
- Enter 1 for population data
- Enter 0 for test data
- Any other entry will run the 
  population data

You will then be asked how many iteration
you wish to run. Please enter a whole
number greater than zero.

========HOW TO CHANGE ALGORITHM========
For Dorling Algorithm:
- Comment line 40 and uncomment line 41

For Bearing augmented Dorling Algorithm:
- Uncomment ine 41 and comment line 40
